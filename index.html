JQuery Range Slider

Yet another JQuery range slider class

View on GitHub Download .zip Download .tar.gz

JQuery Range Slider is a purpose-built slider class, which attempts to recreate the android sliders for web use.

Features

  • Works with mouse events and touch events
  • Can be adjusted through JS
  • Can be disabled dynamically
